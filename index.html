Application Setup Instructions

Before Submitting, please create the following properties in Contact:

  • Unique Identifier
    Object Type: Contact
    Group: Contact Information
    Label: Unique Identifier
    Field Type: Single-line Text
  • Custom Url
    Object Type: Contact
    Group: Contact Information
    Label: Custom Url
    Field Type: Single-line Text
  • Posthog Url
    Object Type: Contact
    Group: Contact Information
    Label: Posthog Url
    Field Type: Single-line Text

Additionally, enter the Posthog Access Token:

To find your Posthog API key or access token, follow these steps:

  1. Log in to your Posthog account.
  2. Navigate to the "Project settings" page.
  3. Find the "API Key" or "Access Token" section.
  4. Copy the provided key or token and paste it into the field above.

If you do not have an API key, you can create one by following these steps:

  1. Log in to your Posthog account.
  2. Navigate to the "Project settings" page.
  3. Click on the "Create API Key" button.
  4. Enter a name for your API key and click "Generate".
  5. Copy the generated API key and paste it into the field above.
